Estructura de páginas web

Tener una buena elaboración de la estructura tiene muchos beneficios, estos son: contar con todos los elementos necesarios para tener una pagina web clara, contar con un banner es algo muy llamativo para las personas que consuman la página web, normalmente un banner en la estructura va en el header, otro beneficio de la elaboración de paginas web es que al crear una pagina web es como una extensión del producto a ofrecer

  • Partes de una página web en función de su estructura
    • Cabecera o header
    • Cuerpo o body
    • Pie de página o footer
  • Partes de una página web en función de la distribución del contenido
    • Inicio o home
    • Contacto
    • Productos y servicios
    • Política de privacidad
